# Sparkie

**AI-powered electrical standards assistant for Australian electricians**

Sparkie is a RAG (Retrieval-Augmented Generation) application that helps electricians quickly access and understand AS3000 electrical standards and compliance documentation.

## Features

- Semantic search across AS3000 electrical standards
- Natural language query interface
- Knowledge graph-enhanced retrieval
- Evaluation framework for continuous improvement

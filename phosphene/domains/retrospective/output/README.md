# Retrospective (Domain)

Primary artifacts:
- `postmortem` — what happened, impact, root causes, action items.
- `playbook` — reusable runbooks / checklists / “how we do X”.

Canonical locations:
- Working artifacts:
  - `phosphene/domains/retrospective/output/postmortems/`
  - `phosphene/domains/retrospective/output/playbooks/`

Handoff (default):
- **From Test Management / Scrum Management** when outcomes are known.
- **To all domains** as process improvements and reusable guidance.
